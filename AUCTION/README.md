# Конкурс для Backend-разработчиков — Аукцион цифровых товаров

## Обзор

Этот проект реализует многораундовую цифровую систему аукционов цифровых товаров, аналогичную Telegram Gift Auctions, разработанную для интеграции в экосистему Mini App @cryptobot. Система обладает защитой от снайпинга, финансовой корректностью, надёжностью параллельности и легким пользовательским интерфейсом, совместимым с руководящими принципами дизайна Telegram.

## Особенности

- Механизм многораундовых аукционов (в каждом раунде вручается один цифровой предмет)
- Защита против снайперов (продление раунда на поздние предложения)
- Финансовая безопасность при операциях с балансом
- Обновление предложений в режиме реального времени и пользовательский интерфейс
- Совместимость с Telegram Mini Apps
- Развертывание готового к работе проекта с Dockerfile

## Стек

- **Backend**: Node.js + TypeScript
- **Database**: MongoDB + Mongoose
- **Frontend**: Vanilla HTML/CSS/JS with Telegram WebApp SDK
- **Deployment**: Docker + Docker Compose

## Как это работает

- Аукционы состоят из нескольких раундов
- В каждом раунде один товар присуждается участнику, предложившему наивысшую цену
- Поздние предложения продлевают раунд (анти-снайпинг)
- Средства заблокированы на торгах и высвобождены соответствующим образом
- Средства победителей передаются создателю аукциона
- Проигравшие получают полное возмещение

## Локальный запуск

```bash
git clone <https://github.com/Natllzzz/auction-backend>
cd auction-backend
docker-compose up --build
```
